              比例线段 #

              对于四条线段 aaa，bbb，ccc，ddd 如果两条线段的比（即它们长度的比）与另两条线段的比相等，如 ab=cd(ad=bc)\frac{a}{b} = \frac{c}{d} (ad=bc)ba​=dc​(ad=bc)，我们就说这四条线段成比例。

              比例性质 #

              ab=cd⇔ac=bd \frac{a}{b} = \frac{c}{d} ⇔ \frac{a}{c} = \frac{b}{d} ba​=dc​⇔ca​=db​

              ab=bc→b2=ac \frac{a}{b} = \frac{b}{c} → b^2 = ac ba​=cb​→b2=ac

              比例中项 #

              如果 aaa、bbb、ccc 三个量成比例即 a:b=b:ca:b = b:ca:b=b:c，bbb 叫做 aaa 和 ccc 的比例中项。

              例子 #

              1:2=2:4 1:2=2:4 1:2=2:4

              222 是 111 和 444 的比例中项。

              定义 #
